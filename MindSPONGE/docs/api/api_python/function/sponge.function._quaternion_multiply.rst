sponge.function._quaternion_multiply
========================================

.. py:function:: sponge.function._quaternion_multiply(tensor_1, tensor_2)

    对给定张量进行四元数乘法操作。

    参数：
        - **tensor_1** (Tensor) - 张量的shape为 :math:`(B, 4)`。
        - **tensor_2** (Tensor) - 张量的shape为 :math:`(B, 4)`。

    返回：
        Tensor。q。张量的shape为 :math:`(B, 4)`。
