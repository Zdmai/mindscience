sponge.function._constant_multiply
======================================

.. py:function:: sponge.function._constant_multiply(tensor_1, constant)

    对给定张量和常量进行四元数乘法操作。

    参数：
        - **tensor_1** (Tensor) - 张量的shape为 :math:`(B, 4)`。
        - **constant** (Tensor) - 张量的shape为 :math:`(B, 1)`。

    返回：
        张量的shape为 :math:`(B, 4)`。
